Usando Parâmetros Rest

O resultado da função soma(a, ...nums) com parâmetros (10, 225, 334, 427) é 796 e pode ser visto também no console.

Função soma() exportada

// calcula a soma dos números passados como argumentos

export function soma(a, ...nums) {
let arg = Number(a);
let result = arg + nums.reduce((acc,
item) => acc + item, 0);
return result
};

Função soma() implementada em outro módulo

import {soma} from './utils.js';

console.log(soma(10, 25, 334, 427)); // 796
