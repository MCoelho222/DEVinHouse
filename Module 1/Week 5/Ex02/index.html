Exercício 2 Semana 5

Estamos apenas executando e imprimindo no console os métodos de Array.prototype map() e filter()

Código

const arrayNumeros = [1, 2, 3, 4, 5, 6, 7, 8, 9]
let arrayQuadrados = arrayNumeros.map(item => item**2);
let majors = arrayQuadrados.filter(item => item > 30);
console.log(majors); // [36, 49, 64, 81]
console.log(arrayQuadrados); // [1, 4, 9, 16, 25, 36, 49, 64, 81]
