Exercício 6 Semana 5

Uma função que concatena arrays utilizando Spread

Código exportado de utils.js

// função recebe arrays e concatena

export function concatena(a, ...arrs) {
let result = [...a];
arrs.forEach(item => {
let arr = item;
result = [...result,...arr];
})
return result;
};

Código importado em app.js e executado

import {concatena} from './utils.js';

console.log(concatena([1, 2, 3], [4, 5, 6], [7, 8, 9])); // [1, 2, 3, 4, 5, 6, 7, 8, 9]
