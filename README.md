## Módulo 1 

Período: 28/02/2022 - 20/05/2022  

### Tópicos

* Fundamentos da programação web: 
* JavaScript e GitHub
* JavaScript
* Projeto 1
* VueJS
* Projeto 2


## Módulo 2

### Tópicos

* Python
* Projeto 1

* Projeto 2
